# 🕹️ C Command Battle AI Challenge

## 1. 프로젝트 개요

이 프로젝트는 C 언어 기반 printf/scanf 콘솔 환경에서 진행되는 **2인용 동시 턴제 전략 게임**이다.  
학생은 외부 AI 파일을 작성해 Player 행동을 제어하며, **퀴즈 시스템**을 통해 고급 스킬을 해금해야 한다.

- **목표:** 상대 HP를 0으로 만들기  
- **맵 크기:** 15 × 7  
- **초기 자원:** HP 5 / MP 5  

---

## 2. 보안 및 AI 등록 규칙

### A. Key 인증 및 API

1. **Key 획득:** `register_player_ai` 호출 시 고유 `reg_key` 부여  
2. **API 인증:** 모든 스킬 관련 API(`attempt_skill_unlock`, `is_skill_unlocked`)는 `reg_key` 인증 필요  
3. **퀴즈 시스템:** 고급 스킬 해금은 `quiz_data.csv`의 정답 문자열 제출

### B. 스킬 잠금 및 페널티

- 고급 스킬은 기본 잠금 상태  
- 잠긴 스킬 사용 시  
  - **스킬 미발동**  
  - **즉시 HP 2 피해**

---

## 3. 턴 진행 흐름

턴은 **동시 입력 → 순차 처리** 방식으로 진행된다.

### 턴 처리 단계

1. **DoT Phase:** 독(Poison)일 경우 HP 1 감소  
2. **우선권 결정:** 50% 확률로 P1 또는 P2가 선공  
3. **Move Phase:**  
   - 기본 이동, 점멸 처리  
   - 충돌 시 **선공권자만 이동 성공**  
4. **Magic Phase:**  
   - 마법/전략 스킬을 **선공 → 후공** 순으로 처리  
5. **Action Phase:**  
   - 물리 공격을 **선공 → 후공** 순으로 처리  
   - 후공은 선공 공격을 맞고 살아 있을 경우에만 반격 가능
